
## How to run

type
```bash
"mvn clean verify -Dtestng.dtd.http=true"
```
in terminal

## Main Features
* Selenium
* Appium 
* Java
* TestNG
* Page Object Model
* [Log4j 2](https://logging.apache.org/log4j/1.2/apidocs/org/apache/log4j/PatternLayout.html)
* Capture Screenshots
* Cucumber BDD Framework
* Cucumber HTML Report
